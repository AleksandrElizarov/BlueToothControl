Создание андроид приложения для управления игрушечной машинкой посредством BlueTooth канала

Одним из первых моих андроид приложений, с которого, можно сказать, я погрузился в "Мир Андроид", стало приложение для управления китайской игрушечной машинкой которая раннее управлялась с помощью джойстика через радиоканал и имела ограниченный функционал команд: вперед - назад - направо - налево. Разрабатываемое приложение на андроиде предоставляет неограниченный выбор техник, настроек, каким образом и как можно передавать команды, а также принимать команды/данные с управляемого модуля.
Создание приложения(назовем его HummerDrive ) производилось в интегрированной среде разработки Android Studio 3.5.3 .
Так как главным пользователем данного приложения является мой сын, то UI разрабатывался наиболее простым в использовании и визуально адаптивным.


И обозначение кнопок:
Пользовательский интерфейс приложения составлен из фрагментов.
Приложение загружается с основного фргамента, представленого выше. Здесь выведены кнопки:

ВПЕРЕД - при нажатии, машина движется вперед с заданной скоростью выставленную с помощью spinner - СКОРОСТЬ, при отжатии машина останавливается.

НАЗАД - при нажатии кнопки, движется назад с заданной скоростью выставленную с помощью spinner - СКОРОСТЬ, при отжатии машина останавливается.

ВПРАВО - при нажатии, передние колеса поворачиваются вправо, при отжатии колёса устанавливаются прямо.

ВЛЕВО - при нажатии, передние колеса поворачиваются влево, при отжатии колёса устанавливаются прямо.

СКОРОСТЬ - с помощью spinner устанавливается скорость с которой будет ехать машина(значения относительные).

ТЕМПЕРАТУРА - значения температуры(в градусах) на радиаторе платы-драйвера электродвигателя L298N

ЗАПРОС ТЕМПЕРАТУРЫ - при нажатии кнопки, приложением отправляется команда, чтобы ардуино с помощью температурного датчика DS18b20 произвел измерения на плате-драйвер и значения температуры отправлены в приложение и выведены на экран.

ФАРКОП - при нажатии кнопки, активи обновляется фрагментом, которое выводит обозначение фаркопа и три положения фаркопа: вверх, нейтраль, вниз, которое может менять при помощи сервопривода.

СОСТОЯНИЕ ПОДКЛЮЧЕНИЯ К BLUETOOTH - в статус-баре выводится информация об успешном подключении BlueTooth модуля телефона с BlueTooth модулем ардуино - HC-05.

ЗВУКОВОЙ СИГНАЛ - при нажатии кнопки, отправляется запрос ардуино на включение активного зуммера, который включается на 400 мсек.

ВКЛ/ВЫКЛ ФАР - кнопка-переключатель, при помощи которой можно включить или выключить питание светодиодов на передних фарах.

Также имеется в тулбаре имеется меню с помощью которого можно просмотреть информацию о Используемой модели Bluetooth модуля и его MAC-адрес а также информацию о приложении.
